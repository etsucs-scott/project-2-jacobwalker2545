
# War Game Simulation

## How to build 
to build this 
dotnet build

## Run Instructions
to run the game
dotnet run --project src/WarGame.Console

## Player Count 
you will be asked for a player count 2-4
this will determine how the game plays out 

## Desription 
This project simulates the card game War using the aspects
Queue (As player hands)
Stack (deck)
list (pot)
Dictionary (player)
